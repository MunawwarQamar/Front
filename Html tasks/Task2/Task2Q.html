Final HTML Task

The project will be a restaurant website, and you need to create the following pages:

  • about.html: A page that talks about the restaurant and includes images.
  • contact.html: A page containing a contact form for the restaurant.
  • register.html: A page for users to create an account.
  • login.html: A page for users to log in.
  • wishlist.html: A page that contains a list of 3 favorite dishes from the restaurant.
  • menu.html: A page that contains the restaurant's menu (each dish should have an image).
  • index.html: A homepage to navigate between all the restaurant pages.

Users should be able to navigate between all pages from any page.

Important Notes:

  • For contact.html: It must contain a form with fields: [ username, email, textarea ].
  • For login.html: It must contain fields: [ email, password ].
  • For register.html: It must contain fields: [ username, email, age, password, gender (radio buttons) ].
  • The img tag must include the alt attribute.
  • The HTML document must contain head and body sections.
  • All images should be stored inside an image folder.
  • Do not use any element that has not been explained very important .
